# Kitty Terminal
## Tabs:

* __Ctrl + Shift + t__ -> new tab

* __Ctrl + Shift + q__ -> close tab

* __Ctrl + Shift + Right / Left__ -> move to next/previous tab

* __Ctrl + Shift + . / ,__ -> move tab forward/backward

## Windows (Splits):

* __Ctrl + Shift + E__nter -> new window

* __Ctrl + Shift + w__ -> close window

* __Ctrl + Shift + [ / ]__ -> move to next/previous window

* __Ctrl + Shift + f / b__ -> move window forward/backward

* __Ctrl + Shift + l__ -> next window layout

## Scrolling & Clipboard:

* __Ctrl + Shift + Up / D__own -> scroll line up/down

* __Ctrl + Shift + PageUp / P__ageDown -> scroll page up/down

* __Ctrl + Shift + h__ -> browse scrollback buffer in pager

* __Ctrl + Shift + c__ -> copy to clipboard

* __Ctrl + Shift + v__ -> paste from clipboard
