# Hyprland

## System & Apps:

* __SUPER + Q__ -> kill active window

* __SUPER + Return__ -> open terminal (kitty)

* __SUPER + E__ -> open file manager (nautilus)

* __SUPER + R__ -> open app launcher (rofi)

* __SUPER + V__ -> open clipboard manager (rofi)

* __SUPER + C__ -> open editor (VS Code)

* __SUPER + W__ -> switch wallpaper

* __SUPER + BACKSPACE__ -> lock screen (hyprlock)

* __SUPER + SHIFT + T__ -> reset touchpad/input

* __SUPER + SHIFT + CTRL + M__ -> exit hyprland

## Window Management (Dwindle):

* __SUPER + J__ -> toggle split

* __SUPER + L__ -> swap split

* __SUPER + K__ -> toggle floating

* __SUPER + Arrow Keys__ -> move focus (left, right, up, down)

* __SUPER + SHIFT + Arrow Keys__ -> move active window around

* __SUPER + LMB__ -> move window (click & drag)

* __SUPER + RMB__ -> resize window (click & drag)

* __ALT + R__ -> enter resize submap [Arrow keys to resize -> Esc to exit]

## Workspaces:

* __SUPER + [0-9]__ -> switch to workspace 1-10

* __SUPER + Scroll__ -> scroll through existing workspaces

* __SUPER + Up / Down__ -> scroll through existing workspaces

* __SUPER + SHIFT + [0-9]__ -> move active window to workspace 1-10

* __SUPER + S__ -> toggle special workspace [spotify]

* __SUPER + SHIFT + S__ -> move window to special workspace [spotify]

* __SUPER + X__ -> toggle special workspace [terminal]

* __SUPER + SHIFT + X__ -> move window to special workspace [terminal]

## Screenshots:

* __Print Screen__ -> full screen to clipboard [notifies on success]

* __ALT + SHIFT + S__ -> select region to clipboard

* __ALT + CTRL + S__ -> select region to editor [swappy]

Media & Hardware:

* __XF86AudioMute__ -> toggle audio mute

* __XF86AudioRaiseVolume__ -> volume +5%

* __XF86AudioLowerVolume__ -> volume -5%

* __XF86MonBrightnessUp__ -> brightness +10%

* __XF86MonBrightnessDown__ -> brightness -10%
