# Neovim
## Window & Pane Management:

* __Ctrl + h/j/k/l__ -> move between splits (Left, Down, Up, Right)

* __Arrows__ -> resize active split

* __<leader> |__ -> split vertical

* __<leader>__ - -> split horizontal

## Navigation:

* __s + <2_chars>__ -> flash jump to any visible text instantly

* __<leader> a__ -> harpoon pin current file to quick-list

* __Ctrl + e__ -> harpoon open quick-menu of pinned files

* __<leader> 1/2/3/4__ -> harpoon teleport to pinned file 1-4

* __<leader> ff__ -> telescope find file by name

* __<leader> fg__ -> telescope find text inside all files (Grep)

## Code & Intellisense:

* __gd__ -> go to definition

* __K__ -> hover (Show docs/types in float)

* __<leader> cr__ -> rename variable/function project-wide

* __Tab / Shift + Tab__ -> navigate autocomplete menu

* __Ctrl + ___ -> toggle floating terminal
